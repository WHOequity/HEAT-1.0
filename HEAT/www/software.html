Software

 

The Health Equity Assessment Toolkit was developed using R: A Language and Environment for Statistical Computing and shiny: Web Application Framework for R. Additional R packages used to support the toolkit include:

  • ggplot2: elegant graphics for data analysis.
  • dplyr: A Grammar of Data Manipulation.
  • RColorBrewer: ColorBrewer Palettes.
  • grid: provides tools to draw and arrange basic shapes.
  • gridExtra: Miscellaneous Functions for "Grid" Graphics.

The standalone version of the toolkit uses a portable version of R and the portable edition of the Mozilla Firefox web browser.
